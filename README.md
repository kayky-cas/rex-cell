# Rex Cell
Rex Cell is a command line csv reader made in rust, that can calculate expressions.

### Objectives
* [ ] Read the csv file input
* [ ] Translate to a text table
* [ ] Convert to number
* [ ] Convert to expression
* [ ] "UI"

#### Expressions
* [ ] sum, sub, mult and div
* [ ] if and else

#### UI
* [ ] nvim
* [ ] navigation
